### 聲韻標記與排版 / Phonological Markers and Typesetting

The classical tonal categories—Level, Rising, Departing, and Entering—possess four primary encoded markers (U+302A-302D). Hollow glyph representations rigorously denote voiceless consonants. Expanding upon this, encountering the Variant Selector (U+FE00) mathematically transforms them into solid glyphs to accurately indicate voiced initial consonants.

These twelve resulting markers are universally enforced with a pure overlay color in hex-red (#FF0000FF).

### 經紀與部署 / File Structure and Deployment

凡茲經紀，皆勒爲文獻。常規字體 JekSans-Regular.otf 及 JekSans-Regular.woff2，曁樣式表 JekSans-Regular_base64.css，咸集於根目錄之下。調用之法，可遠程而致。
All aforementioned configurations are strictly compiled into this repository architecture. The standard typographic binaries `JekSans-Regular.otf` and `JekSans-Regular.woff2`, alongside the embedded stylesheet `JekSans-Regular_base64.css`, are centrally aggregated within the root directory. Implementation protocols explicitly permit remote content delivery network invocation of the Web Open Font Format.

```css
@font-face {
    font-family: 'Jek Sans base64';
    src: url(data:font/woff2;charset=utf-8;base64,...) format('woff2');
    font-style: normal;
    font-weight: normal;
    font-stretch: normal;
    unicode-range:
        U+268A-268B, U+1D300,
        U+268C-268F, U+1D301-1D305,
        U+2630-2637,
        U+1D306-1D356,
        U+4DC0-4DFF,
        U+302A-302F, U+FE00-FE08;
    font-display: swap;
    font-language-override: normal;
    ascent-override: 50%;
    descent-override: 50%;
    line-gap-override: 0%;
}
```
